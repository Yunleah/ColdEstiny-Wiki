# 命令

* 输入 **`coldestiny`** 或 **`ce`** 可查看详细命令结构

```
Usage: /coldestiny
         ├── help
         ├── world
         ├── eval <...>
         └── reload
```

## help

```
/coldestiny help
```

* 获得插件帮助

## world

```
/coldestiny world
```

* 返回被插件成功读取到的世界

## eval

```
/coldestiny eval <script>
```

* **script**: Kether脚本语句
* 执行Kether脚本
